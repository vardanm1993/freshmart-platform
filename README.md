# FreshMart Platform

![Backend CI](https://github.com/vardanm1993/freshmart-platform/actions/workflows/ci-backend.yml/badge.svg)
![Frontend CI](https://github.com/vardanm1993/freshmart-platform/actions/workflows/ci-frontend.yml/badge.svg)

> AI support (Laravel AI SDK) is used for **draft-only suggestions** (human-in-the-loop), never auto-published.

Production-grade delivery platform (v1 single-business). Roadmap includes v2 marketplace vendors, NativePHP courier app, and a deferred mini CRM.

## Stack
- Backend: Laravel (API)
- Frontend: Vue 3 + TypeScript
- Data: PostgreSQL + Redis
- Realtime: Laravel Reverb + Echo
- CI: GitHub Actions
